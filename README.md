# First API Endpoint

This is my first backend API project.

## Endpoints

### GET /
Returns:

```json
{
  "message": "Hello Backend!"
}
```

### GET /about

Returns:

```json
{
  "name": "Adeena",
  "course": "Backend AI Engineering"
}
```

## How to Run

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
node first_task.js
```

3. Open in your browser:

- http://localhost:3000
- http://localhost:3000/about

## Technologies Used

- Node.js
- Express.js
